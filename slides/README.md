# Presentation Slides

30-minute presentation on Poetry GP for CS 2050 class.

## Key Files

- **presentation.md** - Main slides (concise, presentation-focused)
- **FAQ.md** - Detailed companion document (answers deep questions)
- **gp_1d_example.pdf** - 1D GP visualization
- **build_slides.sh** - Build PDF with pandoc

## Structure

1. **Problem Motivation** (3 min, slides 1-3)
2. **GP Method & Computation** (7 min, slides 4-11)
   - Ridge → Dual → Kernels → GP
   - 1D GP example showing uncertainty
   - Posterior update (block covariance structure)
   - Computational bottlenecks highlighted
3. **Active Learning** (5 min, slides 12-13)
   - Exploit vs Explore
   - Acquisition functions and costs
4. **Implementations & Benchmarks** (10 min, slides 14-24)
   - Python, PyBind11, ScaLAPACK
   - **Actual benchmark data** (not expected values!)
   - War stories (environment debugging)
5. **Live Demo** (5 min, slide 25)
6. **Future Work** (slide 26)
7. **Q&A** (slide 27)

## Important Notes

**Slides are concise**: Only key equations, results, tables. Verbose explanations in FAQ.md.

**All benchmarks use actual data**: From `results/large_scale_fit_20260406_233011.csv`

**1D GP example**: Generated by `create_1d_gp_example.py` (run automatically)

## Building Slides

### Prerequisites

Install pandoc if not already available:

```bash
conda install -c conda-forge pandoc
```

Or on Ubuntu:

```bash
sudo apt-get install pandoc texlive-latex-base texlive-latex-extra
```

### Build

```bash
cd slides
bash build_slides.sh
```

This generates `presentation.pdf` using pandoc + beamer.

### View

```bash
evince presentation.pdf
```

## Customization

### Update Benchmark Figure Paths

The slides reference figures from the pedagogical benchmarks:

```markdown
![Fit time vs m](../results/pedagogy_TIMESTAMP/fig_fit_vs_m.png)
```

Before building, update these paths with actual benchmark output directory:

1. Run the pedagogical benchmarks (if not already done):
   ```bash
   sbatch scripts/pedagogical_benchmarks.slurm
   ```

2. Find the output directory:
   ```bash
   ls -td results/pedagogy_* | head -1
   ```

3. Update figure paths in `presentation.md`:
   ```bash
   # Replace pedagogy_* with actual directory name
   sed -i 's|pedagogy_\*|pedagogy_20260407_123456|g' presentation.md
   ```

Alternatively, you can use symbolic links or embed figures directly.

### Adjust Timing

If sections run long/short during practice:

- **Reduce math detail**: Remove some intermediate equations
- **Add more war stories**: Students love debugging stories
- **Adjust demo time**: Can shorten to 3 minutes if needed
